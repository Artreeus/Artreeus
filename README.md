## About

I build full-stack web applications end to end — from typed React/Next.js front ends to Node.js/Express APIs backed by MongoDB and MySQL. At **[BdREN — Bangladesh Research &amp; Education Network](https://www.bdren.net.bd)** (the country's national research and education network), I work on full-stack web applications supporting the organisation and its connected institutions.

Alongside conventional product work, I build AI-assisted tooling: integrating the **Google Gemini API** into product flows, and authoring **Model Context Protocol (MCP)** servers that give AI assistants new, well-scoped capabilities.
